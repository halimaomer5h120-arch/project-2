# Student Registration System

A simple web system to manage student records.

## Features
- *Add:* Easy form to save student details.
- *View:* Table showing all registered students.
- *Delete:* Remove students with one click.

## Components
- *index.php:* Main page and student list.
- *db.php:* Database connection.
- *process.php:* Logic for saving and deleting data.
- *database.sql:* Database structure and table setup.
